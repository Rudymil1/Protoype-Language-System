How to Use the Smart Menu

  1. Tap the microphone button to start
  2. Allow microphone access when prompted
  3. Speak naturally in your preferred language (e.g., “Hello”, “Magandang gabi”, “Konnichiwa”)
  4. Wait for the system to detect your language and switch the menu automatically
  5. Browse the menu or use voice commands to order
  6. If it doesn’t work, use the manual language buttons below
